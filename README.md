# Clock_Tree
To run the program, use the following command:
./ctsRun /path/to/folder

an example:
./ctsRun ./pre_submit/case1

If the folder contains the following files:
/path/to/folder/constraints.txt
/path/to/folder/problem.def
The program will generate an output file solution.def in the same folder.

To display the help information, use the following command:
./ctsRun -h

The source code of our program is located in the 'code' directory in the current folder.
